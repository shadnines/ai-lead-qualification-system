# ai-lead-qualification-system

## Overview

This project automatically analyzes incoming business leads using AI.

The workflow captures leads from Google Forms, sends them to GPT-5 for analysis, categorizes them as Hot/Warm/Cold, updates Google Sheets, and sends email notifications for hot leads.

## Technologies Used

- Relay.app
- OpenAI GPT-5
- Google Forms
- Google Sheets

## Features

- Automatic lead capture
- AI lead scoring
- Hot/Warm/Cold classification
- Automated spreadsheet updates
- Email alerts for high-value leads

## Workflow

Google Form
↓
Google Sheets
↓
Relay
↓
GPT-5
↓
Update Sheet
↓
Email Notification
